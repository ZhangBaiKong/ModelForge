# 模型整合 ModelForge v0.0.1.0

一个主模型调度多个副模型的 AI 协作编排系统桌面应用。

智能拆解任务、分配给最合适的模型执行，省 token，提高命中率。

## 它能做什么

你发一张图片 + 一句话："根据这个界面写出相同的代码"
你 → 主模型（接收任务）
↓
主模型自己分析后，决定：
├── 图片识别 → 交给副模型1（识图能力强）
├── 界面生成 → 交给副模型2（擅长前端）
└── 代码编写 → 自己做（擅长代码）
↓
主模型整合所有结果 → 返回完整答案

text

每个模型只做自己擅长的事，主模型还会为副模型"翻译"出更精确的指令。

## 功能

- 🧠 **智能编排** — 主模型自动拆解任务，分配给合适的副模型
- 🖼️ **多模态** — 图片自动交给识图模型，结果回到主模型整合
- 💬 **多对话管理** — 新建、置顶、搜索、重命名、导出
- ⚙️ **自定义模型** — 自由添加主模型和多个副模型，每个模型可独立配置
- 📊 **运行日志** — 实时显示编排过程，看 token 花在哪了
- 🎨 **主题切换** — 深色 / 浅色
- 🌐 **中英文** — 支持中文和英文界面

配置模型

1.启动后，点击右侧 ＋ 添加模型
2.填写 API 地址、密钥、模型名
3.选择方向（文本对话 / 代码生成 / 图片识别 / 界面生成 ...）
4.点击 测试连接 确认可用
5.保存，选择设为主模型或副模型

开始对话

左侧点击 新建对话
输入文字，Ctrl + Enter 发送
可附加图片（📎 按钮），选中后输入框上方会显示附件提示
右侧日志窗口查看编排过程
