# 模型整合 (ModelForge)

AI 协作编排系统桌面应用 —— 一个主模型调度多个副模型，智能分工，省 token，提高命中率。

## 功能

- 🧠 智能编排：主模型自动分析任务，分配给最合适的副模型
- 🖼️ 多模态支持：图片自动发送给识图模型，结果整合到主模型
- 💬 多对话管理：新建、切换、置顶、搜索、重命名、导出
- ⚙️ 自定义模型：自由添加/编辑主模型和多个副模型
- 📊 运行日志：实时查看编排过程
- 🎨 深色/浅色主题，中/英文界面

## 快速开始

### 1. 安装依赖

pip install -r requirements.txt

### 2. 运行

python main.py

### 3. 配置模型

1. 点击右侧 ＋ 添加模型
2. 填写 API 地址、密钥、模型名
3. 选择方向（文本对话 / 代码生成 / 图片识别 / 界面生成）
4. 点击测试连接确认可用
5. 保存

### 4. 开始对话

- 左侧新建对话
- 中间输入文字，Ctrl+Enter 发送
- 可附加图片
- 右侧日志窗口查看编排过程

## 编排原理

用户输入 → 主模型分析 → 输出JSON任务分配
  ├── 自己完成的部分
  ├── 副模型1: 识图
  └── 副模型2: 做界面
      → 主模型整合所有结果 → 返回用户

## 文件结构

- main.py           入口
- app.py            GUI 应用
- engine.py         编排引擎 + API 调用
- config_manager.py 配置管理
- config.json       运行时配置
- requirements.txt  依赖
- conversations/    对话存储（自动创建）

## 兼容性

- Python 3.8+
- Windows / macOS / Linux
- 兼容所有 OpenAI 格式的 API
